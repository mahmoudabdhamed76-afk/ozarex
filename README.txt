========================================
  أوزاريكس - نظام إدارة التوريدات (Ozarex ERP)
========================================

نظام Node.js خالص (بدون أي مكتبات خارجية) — يستخدم node:sqlite + node:http المدمجين.
يحتاج Node.js نسخة 22 أو أحدث.

========================================
  النشر على Railway (بدون Docker)
========================================

1. ارفع المشروع كله على Railway (أو اربطه بـ GitHub repo).
2. Railway هيكتشف Node تلقائياً عبر Nixpacks (مش Docker).
3. في إعدادات المشروع → Variables، أضف:
   - APP_PATH =            (اتركه فاضي تماماً للدومين المخصص)
   - TZ = Africa/Cairo     (اختياري - التوقيت)
   ملاحظة: Railway بيوفّر PORT تلقائياً، متحطهوش.
4. Deploy. البرنامج هيفتح على الدومين مباشرة (root).

⚠️ مهم: لو كنت ناشر بـ Docker قبل كده، احذف الـ service القديمة
   أو اعمل Redeploy كامل عشان ياخد إعدادات Nixpacks الجديدة.

========================================
  التشغيل المحلي (للتجربة)
========================================

   node backend/server.js
   ثم افتح: http://localhost:8787

   لتغيير المنفذ:   PORT=3000 node backend/server.js

========================================
  البيانات
========================================

- قاعدة البيانات: backend/data/ozarex.db (SQLite)
- على Railway: استخدم Volume مربوط بـ /app/backend/data
  عشان البيانات متتمسحش مع كل deploy.
- نسخ احتياطية تلقائية داخل قاعدة البيانات + تصدير يدوي من الإعدادات.
